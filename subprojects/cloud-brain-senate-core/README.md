# Cloud Brain Senate Core

This is the top-level subproject for:

- `Commander`
- `9 lobster elders`

It is the cloud brain role system inside the same repository.

It is not:

- the full platform
- the edge runtime
- the desktop delivery shell
- every downstream engineering service

Owned scope:

1. role system for `Commander + 9 elders`
2. role definitions and prompts
3. identities and collaboration contracts
4. brain-layer workflow logic
5. Commander allocation logic

Not owned:

1. edge executor implementation
2. TG delivery infrastructure
3. research crawling implementation
4. memory/vector infrastructure
5. video rendering implementation
6. CTI / trust / XAI service implementation
