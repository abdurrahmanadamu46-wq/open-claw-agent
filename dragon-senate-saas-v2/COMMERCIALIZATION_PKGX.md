# 龙虾元老院商业化手册（pkgx 版）

## 核心卖点

1. 4MB 单二进制（pkgx）即可启动边缘执行器。  
2. 客户机器零预装：无需手工安装 Python / Node / Deno。  
3. 版本锁定 + 缓存复用：同一套 Skill 跨机器可复现。  
4. 环境不污染：停用后不新增全局命令（例如 `deno` 仍不可用）。

## 标准交付话术

- “下载交付包后，执行 `dragon dev` 完成环境就绪。”  
- “执行 `dragon-edge ...` 即可接入云端调度。”  
- “所有边缘能力由 `SKILL.md` 下发，云端可热更新。”

## 客户侧交付步骤

1. 获取 `pkgx` 和 `dragon` 脚本（或通过部署脚本自动安装）。  
2. 执行 `dragon dev` 完成一键环境准备。  
3. 执行 `dragon shim` 生成 `dragon-edge` 命令。  
4. 运行 `dragon-edge --edge_id=... --account_token=... --user_id=...` 接入网络。

## 验收项

1. `dragon dev` 可重复执行，第二次显著更快（命中缓存）。  
2. `dragon-edge` 可拉取任务并上报 DM 线索。  
3. 运行完成后，`deno --version` 在宿主机仍不可用（未全局安装）。  
4. 控制台可看到对应租户的任务、线索和计费增量。

## 定价建议（结合当前能力）

1. 基础版：免费 + 限额（低 token / run）。  
2. 专业版：按月订阅，开放更多边缘节点与更高并发。  
3. 企业版：私有 Pantry + 定制 Skill 包 + SLA。
