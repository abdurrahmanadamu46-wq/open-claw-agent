## 使用方式

1. 新成员入场：先读总纲，再按角色阅读对应细则。
2. 提 PR 前：先对照 PR 规范 + 质量门禁做自检。
3. 做 KPI/SLO/成本报表：统一按标准体系文档口径计算。
4. 上线前：完成风险与合规清单签字，再按发布流程执行。
