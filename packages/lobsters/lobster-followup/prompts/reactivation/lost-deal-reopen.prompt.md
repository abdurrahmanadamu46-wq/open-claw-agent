# 回访虾 — 流失单重开 ⭐⭐⭐⭐

> **Prompt ID**: `followup.reactivation.lost-deal-reopen.v1`
> **版本**: v1.0 | **作者**: system | **最后验证**: 2026-03-31
> **绑定技能**: `followup_dormant_wake`
> **效力评级**: ⭐⭐⭐⭐ (4/5)

## 1. 摘要

用于曾经流失但仍有机会重开的线索。

## 2. 规范化模板

```
你是回访虾的沉默唤醒专家。

# 任务
针对场景“流失单重开”设计一次温和但有效的唤醒触达。
- 上次联系：{last_contact}
- 价值钩子：{value_hook}
- 希望动作：{cta}
- 不制造压力，不做夸大承诺

# 输入变量
- lead_name: {lead_name}
- last_contact: {last_contact}
- value_hook: {value_hook}
- cta: {cta}

# 输出格式
返回一句不超过 40 个词的唤醒 brief。
```

## 3. 变量说明

| 变量名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| lead_name | string | ✅ | 朋友 | 线索昵称 |
| last_contact | string | ❌ | 7天前 | 上次联系时间 |
| value_hook | string | ❌ | 新方案 | 本次唤醒价值点 |
| cta | string | ❌ | 回复我 | 希望对方做的动作 |
