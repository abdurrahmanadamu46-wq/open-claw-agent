# Visualizer Skill Gotchas

## visualizer_storyboard
- 分镜不能只追求炫技，必须服务转化目标。
- 每镜时长要现实可拍，不要写成无法执行的镜头表。

## visualizer_ai_prompt
- AI 绘图提示词要写清主体、场景、光线和禁用元素。
- 过长的 prompt 容易自相矛盾，先抓核心风格再扩展。

## visualizer_image_gen
- 封面和主图的文字区域要预留，不要满画面铺细节。
- 商业素材要注意手部、脸部和品牌元素的失真。

## visualizer_cover_design
- 封面标题不要和平台 UI 抢视觉层级。
- 对比色过猛会影响品牌统一性。

## visualizer_digital_human_script
- 数字人口播脚本要口语化，不能照搬长文案。
- 每句字数过长会导致口型和节奏失真。

## visualizer_digital_human_video
- 数字人视频不要滥用夸张动作，容易像低质广告。
- 先统一人设和镜头语言，再批量生成。

## visualizer_video_edit
- 剪辑不要只堆转场，重点是节奏和信息密度。
- BGM、字幕和口播要避免互相打架。

## visualizer_subtitle_gen
- 自动字幕要人工抽检专有名词和数字。
- 字幕切分过碎会影响阅读和节奏。
