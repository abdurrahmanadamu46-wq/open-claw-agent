## 一、林探必须掌握的核心知识

### 1. 平台算法信号库（保底认知）

```
各平台核心推流信号（保底必知）：

抖音：
  完播率 > 点赞率（最核心信号）
  3秒留存率：决定是否进入第一轮推流池（1000人测试）
  评论互动热度：决定是否进入更大推流池
  关注转化：决定账号权重是否提升
  分享率：决定是否进入破圈推流
  时间段：晚6-10点黄金时段，早7-9点次黄金

小红书：
  收藏率 > 点赞率（小红书核心）
  搜索匹配：标题/正文关键词与搜索词匹配
  封面CTR：决定曝光转化
  评论深度：高质量长评论权重高于刷量评论
  图文比视频分发量稳定（但视频粉丝黏性更高）

微信视频号：
  社交裂变权重最高（微信好友点赞带来二次分发）
  时长≥1分钟获得更多推荐
  与公众号联动带来额外推流

B站：
  投币 > 收藏 > 点赞（创作者激励权重）
  完播率对中长视频至关重要
  弹幕密度是热度信号
  充电/会员转化是长期账号权重
```

### 2. 热点识别模式库（核心资产）

```
爆款预判信号（按可靠程度排序）：

强信号（基本确定爆）：
  ① 发布后2小时内评论量超过200条（正常账号均值5-10倍）
  ② 完播率超过35%（行业均值通常15-20%）
  ③ 收藏率超过10%（行业均值2-5%）
  ④ 头部账号（50万以上粉丝）发布且互动强

弱信号（需要观察验证）：
  ① 某话题词在搜索量上升但还没有爆款
  ② 头部KOL私下讨论某个话题（非公开）
  ③ 竞品账号连续发同类内容且在加速

假信号（常见误判）：
  ① 节假日内容爆款 ≠ 常规时间段也爆（节日效应）
  ② 头部账号爆款内容 ≠ 小账号可以复制（权重差异）
  ③ 单平台爆款 ≠ 跨平台也爆（各平台用户差异）

持续热点 vs 一次性热点判断：
  持续热点：跟用户需求/痛点高度相关（美妆技巧/育儿问题）
  一次性热点：事件驱动（新闻/明星事件/节日）
  → 内容策略不同：持续热点做模板，一次性热点快速追投
```

### 3. 行业知识库（需要持续积累）

```
已知行业热点规律（保底）：

餐饮行业：
  时间规律：午饭前10:00-11:30、晚饭前17:00-18:30爆款集中
  内容公式：真实排队场景 > 主厨故事 > 菜品制作过程
  爆款触发点：稀缺感（限量）/ 价格冲击（超值）/ 颜值冲击

美妆/护肤：
  时间规律：晚上8-11点（睡前护肤场景）
  内容公式：before/after对比 > 成分解析 > 使用教程
  爆款触发点：效果反差 / 成分黑科技 / 平替大牌

住宿/旅游：
  时间规律：周末前（周四/五）和节假日前1周
  内容公式：真实环境展示 > 隐藏亮点揭秘 > 入住体验
  爆款触发点：性价比 / 颜值超预期 / 小众秘境

待积累行业（每月至少新增2个行业）：
  母婴/育儿、运动健身、教育培训、科技数码、服装穿搭
  珠宝首饰、汽车、房地产、宠物、法律/金融
```

### 4. 竞品拆解方法（标准流程）

```
竞品拆解5步法：
  Step 1: 记录基础数据（发布时间/账号粉丝/互动数据）
  Step 2: 识别内容结构（开场/主体/收口各用什么方式）
  Step 3: 提炼爆款基因（对照21条gene库）
  Step 4: 分析为什么能爆（V = P×H×R×E×T×I×F评分）
  Step 5: 判断可复制性（哪些能复制，哪些是账号独特资产）

输出格式：
  {
    "contentType": "...",
    "geneIds": [...],
    "V_score_estimate": ...,
    "replicableElements": [...],
    "nonReplicableElements": [...],
    "recommendedAction": "立即复制/参考学习/观望"
  }
```

