## 一、铁钩必须掌握的核心知识

### 1. 意向评分模型（最核心资产）

```
意向评分维度（5分满分）：

维度1：行为强度（最高2分）
  2分：主动私信询问价格/预约/购买方式
  1.5分：在多条内容下反复评论
  1分：评论询价一次，没有后续
  0.5分：点赞/收藏，无评论
  0分：路过看过，无互动

维度2：需求明确度（最高1.5分）
  1.5分：明确说了"需要XX，什么时候/多少钱"
  1分：表达了兴趣但需求模糊
  0.5分：只是好奇，没说有需求
  0分：需求完全不相关

维度3：决策周期信号（最高1分）
  1分：表达了时间紧迫性（"最近就想买"/"下周要用"）
  0.5分：近期有计划（"这个月考虑"）
  0分：长期计划或无时间信号

维度4：渠道来源质量（最高0.5分）
  0.5分：来自私信/直播间
  0.3分：来自评论区（互动成本较高，意向相对真实）
  0.1分：来自主页访问（意向最不明确）

评分等级：
  4-5分：热线索 → 立刻转 followup，24小时内跟进
  2.5-3.9分：温线索 → 48小时内转 followup
  1-2.4分：冷线索 → 进入培育池，定期唤醒
  <1分：无效线索 → 归档，不浪费资源
```

### 2. 各行业购买决策周期（保底知识）

```
不同行业线索的有效期（决策周期）：

快消品（零食/日用品）：
  决策周期：<3天
  特点：冲动消费，热线索必须当天跟进
  过期风险：超过72小时，热线索降温为冷线索

美妆/护肤：
  决策周期：3-14天
  特点：需要研究成分/看评测/比较价格
  跟进重点：发使用前后对比内容+限时优惠触发

餐饮/美食：
  决策周期：1-7天（通常在下次外出时转化）
  特点：地理位置敏感，本地用户转化率高3-5倍
  跟进重点：确认是本地用户，推送到店路线/优惠

美业（美容/美甲/美发）：
  决策周期：7-30天
  特点：需要档期匹配，价格决策较慢
  跟进重点：先发作品集，再推档期，最后推优惠

住宿/旅游：
  决策周期：7-60天（和出行计划绑定）
  特点：需要了解房型/位置/价格，比价平台多
  跟进重点：突出独特性（不在任何平台找到同款）

教育/培训：
  决策周期：14-90天
  特点：高决策成本，需要多次接触
  跟进重点：先给免费资料/试听，降低决策门槛

B2B/企业服务：
  决策周期：30-180天
  特点：多人决策，需要正式方案
  跟进重点：发案例/白皮书，约线上会议
```

### 3. 假意向识别模式（避免资源浪费）

```
常见假意向特征：

薅羊毛用户（高频出现）：
  行为特征：专门找优惠/赠品/免费试用
  识别方法：历史记录里多次询问折扣，但从不成交
  处理方式：标记为"优惠敏感型"，只在真正有活动时联系

竞品探子：
  行为特征：询问的都是竞品敏感信息（价格底线/供应商/技术参数）
  识别方法：问题异常专业，不像普通消费者
  处理方式：标准回复，不透露敏感信息，及时上报

学生/研究者：
  行为特征：问很多细节问题但没有购买信号
  识别方法：自称"做论文""研究用"
  处理方式：简单回复，不浪费 followup 资源

地域不符用户：
  行为特征：对本地化产品/服务感兴趣但在外地
  识别方法：直接问"你在哪个城市"
  处理方式：告知当前无法服务，询问是否有其他城市需求
```

### 4. 线索移交 followup 的标准格式

```
移交格式（必须完整）：
  {
    "leadId": "唯一ID",
    "source": "评论区/私信/直播间/主页",
    "platform": "抖音/小红书/...",
    "username": "用户昵称",
    "contact": "联系方式（如有）",
    "score": 4.2,
    "grade": "热/温/冷",
    "industry": "所属行业",
    "intentType": "购买/预约/咨询/了解",
    "decisionCycle": "预计决策周期",
    "context": "用户已了解什么/问了什么/关心什么",
    "urgency": "高/中/低",
    "followupSuggestion": "建议followup用什么方式开场"
  }

移交原则：
  ✅ 热线索：立刻移交，附上完整上下文
  ✅ 温线索：48小时内移交，说清决策周期
  ✅ 冷线索：整理后批量移交，每周一次
  ❌ 不要移交：假意向、地域不符、无效信号
```

## 二、保底能力

```
✅ 任意评论/私信 → 2分钟内完成初步意向评分
✅ 评分≥4分的热线索 → 立刻启动移交流程（不拖延）
✅ 假意向识别：遇到4种典型假意向立刻标记
✅ 给 followup 的移交包：格式完整，上下文充分
✅ 每周生成线索质量报告（各渠道来源的质量分布）
```

## 三、需要主动学习的内容

```
1. 每季度用成交数据反训练意向评分模型
   → 哪些被评为"热"的实际没成交？为什么？
   → 哪些被评为"冷"的最后成交了？漏掉了什么信号？

2. 学习各行业的购买心理
   → 美妆用户买什么（功效还是颜值还是性价比）
   → B2B客户怎么做决策（谁是真正决策者）

3. 和 echoer 对话：了解评论区的真实意向分布
   → 每100个高意向评论，最后有多少变成热线索

4. 和 followup 对话：哪类线索移交质量最高
   → 我提供哪些信息，跟进效果最好
   → 优化移交格式

5. 研究销售漏斗模型（从曝光到成交各环节转化率）
```
