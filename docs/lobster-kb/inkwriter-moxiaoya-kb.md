### 2. 文案结构模板（7种已验证结构）

```
结构1：PAIN-AGITATE-SOLVE（PAS痛点放大结构）
  开场：描述用户痛点（共鸣）
  中段：放大这个痛点（焦虑）
  收口：给出解决方案（释放）
  适合：美妆/教育/职场内容

结构2：反差对比结构（contrast_verdict_loop）
  开场：before状态（普通/问题状态）
  中段：after状态（改变后结果）
  收口：一句裁判式判断
  适合：变装/改造/产品效果类

结构3：故事带货弧线（story_sell_shot_arc）
  开场：一个真实场景/故事
  中段：问题和解决过程
  收口：产品是解决方案 + CTA
  适合：餐饮/生活方式/服务类

结构4：干货清单结构
  开场："X个方法/技巧/步骤"
  中段：逐条展开，每条有小标题
  收口：总结 + 收藏提示
  适合：教程/攻略/评测类

结构5：口播旁白结构（locked_voiceover_broll_vlog）
  开场：情感共鸣句（"你有没有..."）
  中段：叙述性旁白 + B-roll画面
  收口：哲理感金句 + 轻CTA
  适合：生活vlog/情感/品牌故事

结构6：数字人IP评论结构
  开场：数字人正脸出现，直接给一个观点
  中段：论据展开（数据/案例/对比）
  收口：互动引导（"你怎么看？评论区见"）
  适合：垂直领域知识/商业观点

结构7：动画/图文箴言结构
  开场：一句引发共鸣的核心句
  中段：句式重复+场景变化
  收口：道理点题 + 保存/转发引导
  适合：情绪价值类/励志类
```

### 3. 各行业文案禁忌词（合规红线）

```
美妆/医美行业：
  禁词：最好/第一/最/极/顶级（最高级词汇）
  禁词：根治/治愈/永久/一劳永逸（疗效承诺）
  替换方案：
    "最好" → "超好用的" / "我最爱的"
    "根治" → "改善" / "有效缓解"

食品行业：
  禁词：治疗/药用/疾病名称（非医疗食品不能涉及疾病）
  禁词：纯天然/无添加（需要有检测报告支撑）
  替换方案：
    "无添加" → "配方简单" / "原料纯净"

教育行业：
  禁词：保过/100%通过/保分（效果承诺）
  禁词：学了就会/轻松学（夸大效果）
  替换方案：
    "保过" → "系统学习" / "专业辅导"

金融/投资：
  禁词：保本/无风险/高收益保证
  谨慎：年化XX%、收益率（需要有免责声明）
```

### 4. 用户语言风格库（按平台/受众分类）

```
抖音用户语言特征：
  年轻化、口语化、强调情绪
  关键词：真的/绝了/离谱/笑死/太香了/yyds
  句式：短句为主，每句不超过15字
  语气：朋友聊天感，不要书面语

小红书用户语言特征：
  精致感、分享感、带入感
  关键词：宝藏/好用到哭/种草/拔草/hin好看/姐妹们
  句式：温柔亲切，可以带表情符号
  语气：闺蜜分享感，有个人体验

B站用户语言特征：
  有深度、有态度、专业感
  关键词：恕我直言/说句公道话/干货/硬核
  句式：逻辑清晰，可以有长句
  语气：专家分享感

视频号/微信用户语言特征：
  朴实真诚，接地气
  关键词：真实/分享/对比/推荐给身边人
  语气：熟人分享感，像发朋友圈
```

## 三、需要主动学习的内容

```
1. 每周分析3-5个高互动评论区（用户在说什么？他们怎么说？）
   → 把发现的新词汇/句式加入语言风格库

2. 每月研究1个行业的Top文案账号
   → 拆解其标题公式+正文结构+CTA方式

3. 向 abacus 要文案CTR数据（哪类标题公式效果最好）
   → 用数据验证直觉，迭代公式库

4. 跟踪平台合规政策（各平台违规词会定期更新）
   → 至少每季度更新一次禁忌词库

5. 和 visualizer 做"文案→视觉转化"联合复盘
   → 搞清楚哪类文案方向最容易出好视觉
```
