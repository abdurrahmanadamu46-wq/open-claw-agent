## 一、苏思必须掌握的核心知识

### 1. 内容策略模板库（保底必备）

```
已验证的行业策略打法（来自 auto workflaw 行业覆盖）：

餐饮行业策略（三板斧）：
  第一斧：真实流量证明（排队/忙碌/全国客户）
    → 对应战术：t_owner_overwhelm_signal + t_nationwide_demand_proof
  第二斧：产品颜值冲击（菜品特写/制作过程）
    → 对应战术：t_finished_form_payoff + t_progressive_build_reveal
  第三斧：低门槛CTA（评论区问地址/发优惠券）
    → 对应战术：t_soft_social_cta + t_comment_pull_cta_close

美妆/美业策略（三段式）：
  开场：痛点共鸣（"你是不是也有这个问题"）
  中间：解决方案展示（before/after对比）
  收口：价值锚定+行动引导

B2B/服务业策略：
  建立权威 → 展示案例 → 给信任背书 → 轻CTA
  关键：B2B用户决策周期长，内容要有"收藏价值"

母婴/育儿策略：
  情感共鸣开场 → 实用干货主体 → 妈妈社群CTA
  关键：妈妈群体收藏率高，内容要有保存价值

教育/培训策略：
  痛点钩子（"这个错误90%的家长都犯过"）
  → 解法预告 → 方法论主体 → 免费资料CTA
```

### 2. 策略公式体系（V = P×H×R×E×T×I×F）

```
每次设计内容策略时，必须对7个维度做评估：

P（Problem Fit 问题适配）：
  这个内容解决了目标用户的什么真实痛点？
  痛点越真实越痛，P值越高

H（Hook Power 钩子力）：
  前3秒能抓住多少比例的目标用户？
  用什么钩子：问题/反差/惊喜/权威

R（Retention Momentum 留存动力）：
  用户有多少理由看完全程？
  每30秒要有一个新的理由继续看

E（Emotional Curve 情绪曲线）：
  内容是否有张力→释放→信任→行动的完整情绪弧？
  不能情绪平（=用户不会收藏/分享）

T（Trust Anchor 信任锚点）：
  用什么方式让用户相信？真实场景/数据/KOL背书

I（Interaction Transfer 互动转移）：
  内容如何引导用户从"看"变成"做"（评论/收藏/购买）？

F（Format Fit 格式匹配）：
  这个内容适合什么格式？（短视频/图文/直播）
  适合哪个平台？（抖音/小红书/视频号）
```

### 3. 文案裁判标准（苏思的决策权限）

```
文案方向采纳决策流程（苏思是决策者）：

收到 inkwriter 的3个版本后，苏思需要评判：

评判维度（按权重排序）：
  1. 方向对不对（40%权重）：是否符合本次策略方向
  2. 钩子够不够强（30%权重）：前3秒能不能抓住人
  3. 平台风格匹配（20%权重）：语言/调性是否符合目标平台
  4. 行动引导清不清（10%权重）：结尾有没有推动行动

采纳等级：
  A级：4个维度都满足 → 直接采纳
  B级：方向对但执行有瑕疵 → 给具体改法，让 inkwriter 修改
  C级：方向偏了 → 说清楚为什么偏，重新brief

苏思不应该做的事：
  ✗ 直接改文案（那是 inkwriter 的工作）
  ✗ 只说"不好"，不说为什么
  ✗ 忽略 abacus 给的数据反馈（数据是最终裁判）
```

### 4. A/B实验设计模板

```
每次A/B实验的标准设计：

实验假设格式：
  "我们假设：{变量X}的变化，会让{指标Y}提升{Z%}"
  示例："我们假设：标题加入数字（如'3个方法'），会让点击率提升15%"

控制变量原则：
  每次只改一个变量，其他变量保持不变
  如果改了多个变量，就不知道是哪个起了作用

最小显著样本量：
  快消品/餐饮：至少发布72小时，获得5000+曝光
  服务业：至少发布1周，获得2000+曝光
  B2B：至少发布2周，获得1000+曝光

实验结论判定：
  有效：实验组比对照组高出15%以上（且不是随机波动）
  无效：差异<10%，判定为无显著差异
  负效：实验组比对照组低10%以上，立即停止
```
