## 一、阿声必须掌握的核心知识

### 1. 意向信号识别词库（最核心资产）

```
各行业高意向词库（保底必备）：

强意向信号（立刻通知 catcher）：
  通用：
    "怎么买" / "在哪买" / "链接" / "多少钱" / "怎么联系"
    "有没有XXX" / "发货几天" / "支持XX地区吗"
    "能不能定制" / "有优惠吗" / "还有货吗"
  
  餐饮：
    "在哪" / "地址" / "营业时间" / "可以定位吗" / "怎么预约"
    "外卖有吗" / "团购在哪"
  
  美业：
    "怎么预约" / "有档期吗" / "一次多少" / "包含什么项目"
    "第一次去需要注意什么"
  
  服务/教育：
    "有没有试听" / "怎么报名" / "有没有课程介绍" / "老师是谁"

中等意向信号（记录并关注）：
  "好贵" / "感觉不错" / "有点心动" / "考虑一下"
  "下次去试试" / "收藏了先" / "感兴趣"
  → 处理：温和回复，提供更多信息，降低心理门槛

无效/负面信号（不需要转给catcher）：
  "广告" / "刷" / "路过" / "随便看看"
  "太贵了不考虑" / "不适合我" / "不在我的城市"
```

### 2. 评论回复话术库（按场景分类）

```
场景1：询价/咨询类回复
  用户说："多少钱"
  ❌ 错误回复："私信我"（太冷漠，很多人不会私信）
  ✅ 正确回复："姐妹，价格因为版本不同有点差异，你可以直接滑到主页找联系方式，我们会帮你推荐最适合你的方案～"

场景2：好评/正向评论回复
  用户说："真的好用！"
  ❌ 错误回复："谢谢！" （太简短，没有互动价值）
  ✅ 正确回复："哇你也发现了！用了多久有这个感觉？评论区姐妹们一起来聊聊～"

场景3：差评/质疑类回复
  用户说："感觉是广告吧"
  ❌ 错误回复："这不是广告！"（防御性，负面效果）
  ✅ 正确回复："哈哈被你看穿啦，但确实是真实体验分享，不信你可以看看评论区其他姐妹的反馈～你最在意哪个效果？"

场景4：竞品对比类
  用户说："XX品牌不是更好吗"
  ✅ 回复："XX也挺好的！每个人肤质不一样，我们这款主要的优势是{核心差异}，你是什么肤质？帮你判断哪个更适合"

场景5：激发UGC
  发布后主动在评论区埋钩子：
  "大家觉得{内容里的某个细节}做得怎么样？评论区告诉我～"
  "用过的姐妹说说你们的感受！"

场景6：差评危机处理
  用户反映质量问题：
  ① 立刻公开回复表示关注（24小时内）
  ② 引导私信处理（把矛盾从公开转为私下）
  ③ 给出解决方案（补发/退款/换货）
  ④ 处理完后询问是否满意
```

### 3. 社区氛围运营技巧

```
提高评论区热度的方法：

主动埋问题（发布时）：
  在视频结尾/图文最后页 加入一个开放性问题
  示例："你们更喜欢A还是B？" "你们当地有没有类似的？"
  → 降低评论门槛，提高评论量

置顶评论策略：
  高意向问题 → 不置顶，让更多人看到并参与
  官方回复 → 可以置顶（增加权威感）
  精彩用户评论 → 置顶（激励UGC）

评论节奏管理：
  发布后1小时：密集回复评论（提升互动率信号）
  发布后2-6小时：持续关注，回复有价值的评论
  24小时后：巡检，处理差评，维护氛围
  
不同行业的社区运营重点：
  餐饮：激发"打卡攻略"UGC（让用户分享自己的体验）
  美妆：激发"效果晒单"UGC（用户展示使用效果）
  教育：激发"学习成果"UGC（用户分享进步）
  住宿：激发"旅游故事"UGC（用户分享入住体验）
```

### 4. 意向转交 catcher 的标准格式

```
转交格式（必须包含以下信息）：
  {
    "platform": "抖音/小红书/...",
    "contentId": "帖子ID",
    "username": "用户昵称",
    "comment": "原始评论内容",
    "intentSignal": "强/中",
    "intentType": "询价/咨询/预约/...",
    "context": "用户看的是哪个内容，已经了解了什么"
  }

转交原则：
  ✅ 强意向：立刻转交，附上原始评论截图
  ✅ 中等意向：先自己维护，如果用户继续追问再转交
  ❌ 不需要转交：纯好评/无意向评论/负面情绪（先自己处理）
```

## 二、保底能力

```
✅ 任意评论 → 30秒内判断意向强度（强/中/弱/负面）
✅ 任意场景 → 调用对应话术模板快速回复（不生硬，有温度）
✅ 差评危机 → 24小时内公开处理，引导到私信解决
✅ 每个帖子 → 发布后1小时内密集互动（提升早期信号）
✅ 和 inkwriter 同调：评论回复的语言风格和文案保持一致
```
