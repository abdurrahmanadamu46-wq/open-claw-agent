## 一、算无遗策必须掌握的核心知识

### 1. 行业基准数据库（保底必备）

```
各平台各行业基准指标（保底认知，持续更新）：

抖音行业基准：
  完播率：
    <15s视频：行业均值35-45%（好内容>55%）
    15-60s视频：行业均值25-35%（好内容>45%）
    >60s视频：行业均值15-25%（好内容>35%）
  
  CTR（封面点击率）：
    餐饮：2-5%（好封面>8%）
    美妆：3-6%（好封面>10%）
    教育：2-4%（好封面>6%）
  
  互动率（评论+点赞+分享）/曝光：
    行业均值：1-3%（好内容>5%）

小红书行业基准：
  收藏率：行业均值2-5%（好内容>8%）
  点赞率：行业均值3-8%（好内容>12%）
  评论率：行业均值0.5-2%（好内容>3%）

转化漏斗基准（内容到成交）：
  曝光→点击：2-8%（取决于封面）
  点击→停留>10s：40-60%（取决于前3秒）
  停留>10s→互动：5-15%（取决于内容质量）
  互动→私信：1-5%（取决于CTA设计）
  私信→成交：10-30%（取决于 followup 质量）
```

### 2. ROI归因模型（核心分析工具）

```
多触点归因模型（标准分析流程）：

用户旅程：
  曝光 → 点击 → 观看 → 互动 → 咨询 → 成交

各环节问题诊断：

曝光低 → 检查：
  发布时间是否在黄金时段（找 dispatcher）
  账号权重是否正常（找 dispatcher）
  标签/话题是否匹配（找 inkwriter）

点击率低（封面CTR低）→ 检查：
  封面视觉冲击力（找 visualizer）
  标题钩子是否够强（找 inkwriter）
  是否使用了 hero_cover_decisive 基因（找 visualizer）

完播率低 → 检查：
  前3秒钩子（对照 g_three_second_hook 基因）
  内容节奏（镜头密度是否符合 g_mechanical_shot_density）
  情绪曲线（是否有 g_emotional_rollercoaster）

互动率低 → 检查：
  CTA设计（是否用了 g_tail_cta_comment_pull）
  是否有讨论触发点（g_comment_transfer_trigger）
  评论区维护（找 echoer）

咨询率低 → 检查：
  意向词覆盖（找 echoer 确认意向词库是否匹配）
  内容信任锚点（T维度：是否有真实场景/数据证明）

成交率低 → 检查：
  跟进速度（找 followup：热线索是否及时跟进）
  话术匹配度（找 followup：话术是否匹配用户痛点）
```

### 3. A/B实验验证标准

```
实验结论判定流程：

Step 1: 确认实验设计是否规范
  □ 只改了一个变量吗？
  □ 对照组和实验组的其他条件相同吗（发布时间/账号/行业）？
  □ 样本量是否足够（至少达到最小显著样本量）？

Step 2: 计算统计显著性
  简化判断规则：
    实验组 vs 对照组，关键指标差异：
    <10%：无显著差异，实验无效
    10-25%：有一定差异，建议扩大样本再验证
    >25%：显著差异，可以得出结论

Step 3: 输出结论
  结论格式：
    "实验假设：{假设内容}"
    "实验结果：实验组{指标}为{X}，对照组为{Y}，差异{Z%}"
    "结论：{支持/不支持}假设，置信度{高/中/低}"
    "建议行动：{继续用A/继续用B/还需要更多数据}"

常见的A/B实验类型：
  标题公式对比（哪类钩子效果更好）
  发布时间对比（上午 vs 晚上）
  封面风格对比（大字版 vs 产品图版）
  CTA方式对比（硬CTA vs 软CTA）
  内容长度对比（15s vs 60s）
```

### 4. 数据洞察输出格式

```
每月给团队的数据洞察报告格式：

报告结构：
  1. 本月整体表现（和上月对比）
  2. 表现最好的5条内容（共同点是什么）
  3. 表现最差的5条内容（根因是什么）
  4. 发现的关键洞察（3条，每条附有数据支撑）
  5. 下月建议行动（给 strategist 的具体建议）

洞察描述格式（必须有数据支撑）：
  ❌ 错误："这类内容效果好"
  ✅ 正确："带反差对比的内容（contrast_verdict_loop结构），
           CTR比普通内容高32%（均值8.2% vs 6.2%），
           完播率高18%（均值38% vs 32%），
           建议：下月增加此类内容比例至40%"

给不同龙虾的定向洞察：
  → inkwriter："这类标题公式CTR最高"
  → visualizer："这类封面风格完播率最高"
  → dispatcher："这个时间段发布ROI最好"
  → strategist："这类内容策略转化率最高"
  → followup："这类线索成交率最高"
```

## 二、保底能力

```
✅ 任意内容数据 → 30分钟内完成基础分析（和行业基准对比）
✅ 内容失败 → 能定位是漏斗哪个环节出了问题
✅ A/B实验结果 → 能给出有数据支撑的结论（不是"差不多"）
✅ 每月数据洞察报告 → 格式完整，有具体行动建议
✅ 给 strategist 的洞察：能翻译成策略语言（不只是数据）
```

## 三、需要主动学习的内容

```
1. 持续更新各行业基准数据库
   → 每季度收集最新的行业基准数据
   → 基准数据来源：蝉妈妈/新榜/灰豚数据/官方报告

2. 学习统计基础（A/B实验的置信度计算）
   → 了解显著性检验的基本原理
   → 避免"小样本得大结论"的错误

3. 研究各平台的数据定义差异
   → 抖音的"播放量"和小红书的"曝光量"不是同一个概念
   → 跨平台比较时要先统一定义

4. 和 strategist 深度对话：洞察如何转化为策略行动
   → 数据分析师的价值不在分析，在于推动行动

5. 学习内容生产知识（懂一点文案/视觉）
   → 能看出"CTR低的原因是封面太暗"，而不只是"CTR低"
```
