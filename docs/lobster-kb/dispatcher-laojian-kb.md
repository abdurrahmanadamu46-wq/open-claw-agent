## 一、老将必须掌握的核心知识

### 1. 平台发布时间窗数据库（保底必备）

```
各行业×各平台 最优发布时间（持续积累）：

抖音发布时间规律：
  黄金时段：工作日 18:00-22:00（完播率最高）
  次黄金：工作日 07:00-09:00（通勤时间）
  周末：14:00-17:00（下午休闲时段）
  
  行业细分：
    餐饮：10:30-11:30（午饭前） + 17:00-18:30（晚饭前）
    美妆：20:00-23:00（睡前护肤场景）
    母婴：09:00-11:00（宝宝睡觉，妈妈刷手机）
    教育：21:00-23:00（成年学习者晚间）
    科技：12:00-14:00（午休）+ 21:00-23:00

小红书发布时间规律：
  黄金时段：工作日 18:00-20:00
  次黄金：周末 10:00-12:00
  
  行业细分：
    美妆/护肤：21:00-23:00（睡前种草）
    美食/餐饮：10:00-12:00 + 16:00-18:00
    旅游/住宿：周末全天，周五晚高峰
    母婴：09:00-11:00

视频号发布时间规律：
  黄金时段：19:00-21:00（朋友圈打开频率高）
  周末：10:00-12:00（周末浏览习惯）
  注意：视频号依赖社交传播，发布时间影响相对较小
```

### 2. 多账号发布策略（防风控规律）

```
多账号矩阵发布规则（关键！）：

账号关联风险规避：
  同一设备：不能同时登录超过3个账号（实测安全上限）
  发布间隔：同IP下多账号发布间隔≥30分钟
  内容差异：同一内容不能完全相同（至少修改封面+标题）
  行为节奏：每个账号每天发布上限：抖音3条，小红书2条

账号健康指标（需要持续监控）：
  绿色：正常推流，无异常
  黄色：流量明显下降，可能触发软限流
    → 处理：降低发布频率，提高内容质量，等待3-7天
  橙色：部分内容不推荐，审核严格
    → 处理：停止发布3天，申诉异常内容
  红色：账号封禁/限流
    → 处理：立刻停止，联系平台客服，等待解封

已知触发风控的操作：
  ① 短时间内大量关注/取关
  ② 发布内容后立刻删除再发（反复操作）
  ③ 评论区批量使用相同话术
  ④ 账号注册时间<7天就大量发布
  ⑤ 同IP下多账号同时频繁操作
```

### 3. 各平台格式规格库（保底）

```
抖音：
  视频：竖屏9:16，1080×1920，时长15s/60s/3min/10min，码率≥2Mbps
  封面：1:1或16:9，禁止色情/政治/违规内容
  字幕：建议内嵌字幕（完播率更高）

小红书：
  图文：最多9张，主图建议3:4竖图，最小800×1000px
  视频：9:16或3:4，时长不超过15分钟
  标题：字数≤20字（显示完整），正文≤1000字

视频号：
  视频：9:16，时长6s-10min，文件≤1GB
  封面：9:16或3:4，清晰度≥720p

B站：
  视频：16:9，建议1080p，时长1min-4h
  封面：16:9，1146×717px，大字标题效果最好

快手：
  视频：9:16，时长7s-15min，码率≥1Mbps
```

### 4. 合规预检清单（发布前必须过）

```
发布前合规自检（10秒快速过）：
  □ 标题是否含有平台禁词（最/第一/特效/治疗等）
  □ 内容是否涉及政治/宗教/敏感人物
  □ 产品是否有明显虚假宣传
  □ 图片/视频水印是否干净（无竞品水印）
  □ 账号最近是否有异常警告（黄/橙色状态）
  □ 本账号今日发布次数是否超过上限

如果有任何一项不确定 → 先暂缓，找 inkwriter/visualizer 确认
```

## 二、保底能力

```
✅ 任意内容包 → 按最优时间窗排期发布
✅ 多账号矩阵 → 安全发布策略，不触发关联风控
✅ 格式自检：收到内容立刻检查是否符合目标平台规格
✅ 账号健康监控：每日巡检，异常第一时间上报
✅ 发布失败率≤2%（格式/时间/账号三重保障）
```

## 三、需要主动学习的内容

```
1. 持续积累各行业×各平台的时间窗数据
   → 每次发布后记录时间+互动表现，逐步优化时间窗判断

2. 跟踪各平台算法和政策更新
   → 至少每月检查一次官方运营公告

3. 学习各平台风控规律（从已有被封案例中总结）
   → 每次账号出问题必须记录原因和恢复方案

4. 和 abacus 对齐数据：哪个时间段发布ROI最高
   → 用数据优化时间窗选择，不只靠经验

5. 和 visualizer 对齐格式规格
   → 每季度更新一次平台格式库
```
