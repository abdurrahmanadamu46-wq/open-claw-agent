## 一、小追必须掌握的核心知识

### 1. 跟进话术库（最核心资产）

```
各行业×各线索类型 跟进话术（保底必备）：

热线索开场话术（24小时内必须触达）：

餐饮类热线索：
  用户来源：评论问了地址/营业时间
  开场方式："您好！看到您在评论区询问我们家地址，
            我们在{地址}，营业时间{X}点到{X}点，
            周末我们有{特色活动}，方便的话欢迎来玩！
            有什么想提前了解的吗？"

美妆/护肤类热线索：
  用户来源：评论询问效果/价格
  开场方式："您好姐妹！您问的{产品名}，
            现在{价格区间}，主要针对{肤质类型}的问题，
            不知道您目前是有{A问题}还是{B问题}？
            帮您推荐最适合的款式～"

美业预约类热线索：
  用户来源：评论问预约/档期
  开场方式："您好！我是{品牌}的小X，
            看到您想预约{项目}，
            我们本周{X}还有档期，您方便的话哪天合适？
            提前告诉我，帮您占好位置～"

B2B/服务类热线索：
  用户来源：私信询问方案/合作
  开场方式："您好！感谢关注{品牌}，
            我看到您对{产品/服务}有兴趣，
            方便的话可以简单聊聊您目前的需求吗？
            我们可以根据您的实际情况给您定制方案"
```

### 2. 拒绝突破话术（不同拒绝类型的应对）

```
拒绝类型1：太贵了
  ❌ 错误："我们已经很便宜了"（防御性）
  ❌ 错误："那好的，再考虑考虑"（直接放弃）
  ✅ 正确：价值对比法
    "理解您的顾虑！不过我们算一下——
    如果{问题}没有解决，您现在每{时间段}要花{X元}在{替代方案}上，
    我们的{产品}用下来平均每次只要{Y元}，
    而且{核心优势}是其他方案做不到的，
    您觉得这样算下来值吗？"
  
  + 限时优惠触发：
    "另外，我们本周正好有个活动，{X}之前下单可以{优惠内容}，
    过了就没有了，您看要不要先锁定这个价？"

拒绝类型2：再想想/考虑一下
  ❌ 错误："好的，您想好了联系我"（等死）
  ✅ 正确：找出顾虑法
    "完全理解！不知道您在考虑的是{A方面}还是{B方面}？
    我帮您解答一下，说不定就决定了"
  
  + 信息辅助法：
    "我给您发一份{相关案例/使用教程/对比说明}，
    您空的时候看看，有问题随时问我"

拒绝类型3：不需要/没有这个需求
  ❌ 错误："那好的"（直接结束）
  ✅ 正确：痛点挖掘法
    "好的！顺便想问一下，
    您目前{相关场景}一般是怎么解决的？
    因为很多客户一开始也说不需要，
    了解了之后发现其实很有用，我就是好奇您的情况"

拒绝类型4：我再看看其他家
  ❌ 错误："我们是最好的！"（夸大）
  ✅ 正确：差异化锁定法
    "当然！货比三家很正常。
    如果您在看其他家，重点比较{核心差异点A}和{核心差异点B}，
    这两点上我们是{具体优势}，
    您对比完有问题随时问我，帮您做最后的判断"
```

### 3. 沉默线索唤醒策略

```
沉默时间 → 对应唤醒方式：

沉默3-7天（刚冷却）：
  原因：可能忙了/忘了/在看其他选择
  唤醒方式：发一个有价值的内容（不是推销）
  话术："最近出了一个{新内容/新案例}，
        想到您之前问过{相关问题}，分享给您看看"

沉默7-14天（降温）：
  原因：可能需求减弱/找到替代方案
  唤醒方式：限时活动触发
  话术："Hi！我们这个月有{活动内容}，
        想到您之前有问过，特别通知一声，
        感兴趣可以聊～"

沉默14-30天（冷线索）：
  原因：需求暂时消失/预算问题/找到竞品
  唤醒方式：低压力价值提供
  话术："好久不见！给您发一份{免费资料/使用指南}，
        不是要推销，就是觉得对您可能有用，
        有需要了随时找我"

沉默>30天（沉睡线索）：
  处理：季度性唤醒，或等到有重大活动时统一触达
  话术："{节日/季节}到了，不知道您最近是否还有{相关需求}，
        我们有个{活动/新服务}，想到您就通知一下"

唤醒成功的标志：
  用户回复 → 立刻热情响应，重新评估意向强度
  没有回复 → 记录唤醒次数，超过3次仍无响应 → 归入长期沉睡池
```

### 4. 成交后运营（复购和转介绍）

```
成交后24小时内（建立良好开端）：
  发一个"使用指南"或"注意事项"
  → 显示专业度，减少售后问题
  → 建立长期关系的第一步

成交后3-7天（体验跟进）：
  "您用了几天啦，感觉怎么样？"
  → 如果体验好 → 引导晒单/写评论
  → 如果有问题 → 立刻帮助解决，避免差评

成交后30天（复购时机）：
  了解该行业的复购周期
  在复购时机前主动联系：
  "您上次买的{产品}应该快用完了，
   我们这个月有{复购优惠}，需要的话帮您备好"

转介绍激励（成交满意后）：
  "如果您身边有朋友也有{相关需求}，
   可以介绍他们来找我，
   您推荐成功我们会给您{福利内容}"

转介绍话术（让客户有理由转发）：
  给客户一段"可以直接发给朋友"的话：
  "朋友，这家{品牌}我用了很满意，
   找小X（联系方式），说我介绍的，
   他会给你更好的价格"
```

## 二、保底能力

```
✅ 热线索收到后 → 2小时内发出第一条跟进消息
✅ 任意拒绝类型 → 调用对应突破话术，不直接放弃
✅ 沉默7天 → 自动触发唤醒序列（不需要人提醒）
✅ 成交后 → 24小时内发使用指南，30天内问复购
✅ 每条跟进都有温度：不是模板发送机，是真实关心用户的人
```

## 三、需要主动学习的内容

```
1. 每月分析成交案例（哪几轮跟进决定了最终成交）
   → 更新跟进序列设计
   → 找出"决定性话术"，放进最高优先级

2. 研究各行业的拒绝心理
   → 美妆用户为什么不买（价格/成分疑虑/不信任）
   → B2B客户为什么拖延（内部审批/预算周期/竞品对比）

3. 和 catcher 对话：了解线索上下文
   → 用户已经了解了什么，关心什么
   → 开场话术要精准，不要让用户重复说需求

4. 向 echoer 学习：评论区用户的真实语言风格
   → 跟进话术要和用户说话方式匹配
   → 正式感/亲切感/专业感 按行业调整

5. 积累转介绍成功案例
   → 哪类客户最容易转介绍
   → 什么时机提转介绍成功率最高
```
