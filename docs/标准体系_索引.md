# ClawCommerce 标准体系索引

> **总负责人建立**。《项目最终交付标准手册 v1.0》是总纲，本目录下的「标准体系_*」文档是事无巨细的可执行规范，全团队必须对照执行。

## 使用方式

1. 新成员入职：先读《项目最终交付标准手册 v1.0》，再按角色阅读上表对应细则。
2. 提 PR 前：必读《PR 与合并规范》与《质量门禁细则》，并按 DoD 自检。
3. 做 KPI/成本/监控报表：按《商业化 KPI 定义与测算规范》《成本与线索测算规范》《SLO 与监控指标定义》执行。
4. 上线前：完成《风险与合规清单模板》签字，并按《发布与变更管理》打 tag 与发布。
