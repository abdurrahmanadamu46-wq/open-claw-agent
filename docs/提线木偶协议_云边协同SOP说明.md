# 提线木偶协议 (Marionette Protocol) — 云边协同 SOP 说明

**高级虾（云端 AI 内阁）与干活虾（客户本地物理龙虾）之间沟通的「加密语言」。**

- 云端只有「脑子」没有「手脚」，本地只有「手脚」没有「脑子」。
- 云端下发结构化 **SOP 指令包**（本目录下 JSON），本地严格按步执行并回传战报。
- 协议版本：`marionette/v1`。

## 类型定义位置

- **TypeScript 类型**：`backend/src/gateway/marionette-protocol.ts`  
  - `MarionetteSopPacket`：完整行动准考证  
  - `MarionetteSopStep`：单步动作（DOWNLOAD_ASSET / NAVIGATE / INPUT_TEXT / UPLOAD_VIDEO / CLICK_SELECTOR / WAIT / REPORT_BACK 等）  
  - `HumanLikeBehavior`：真人化参数（打字速度、步骤间随机延迟）

---

## 五步协同中的位置

1. **战前侦察**：爆款雷达 → 干活虾发「侦察包」（NAVIGATE + GRAB_SOURCE + REPORT_BACK）。
2. **军工厂造弹**：黄金编剧、视觉导演在云端生成文案与视频，**不经过本协议**。
3. **司令部排兵布阵**：算法排期官生成本 SOP 包，塞进队列，指定 `targetNodeId` 与 `scheduledAt`。
4. **前线冲锋**：到点后干活虾按 `steps` 顺序执行，并遵守 `humanLike`（如 60 字/分钟）。
5. **打扫战场与收割**：干活虾监控评论区，把「求链接」等话术回传；转化黑客/金牌销冠在云端决策收割。
