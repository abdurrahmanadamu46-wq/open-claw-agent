## 二、一键启动（How to Run）

### A. 只给客户看演示（最快）

```powershell
cd web
npm install
npm run dev
```

浏览器打开：**http://localhost:端口/demo.html**（端口以 `npm run dev` 终端里 **Local** 显示为准，若 3000 被占用会变成 3001、3002…）；见 `web/DEMO_怎么跑.md`。

### B. 总控后端 + Redis（扫码或 VIP 连调）

```powershell
# 终端 1：Redis（或用 Docker）
docker run -d -p 6379:6379 redis:7-alpine

# 终端 2：Nest backend
cd backend
npm install
$env:JWT_SECRET="your-secret"
npm run start:dev
```

API：`POST /api/v1/devices/bind-ticket`  
WS：`/agent-cc`（Socket.io），详见 `backend/README.md`。

### C. VIP 客户端（一号客户 exe 前体）

```powershell
cd 仓库根目录
copy scripts\vip-build\.env.vip.example scripts\vip-build\.env.vip
# 编辑 .env.vip
npm run vip:run
```

打 exe：`npm install -D pkg` 后 `npm run vip:pkg` → `dist/vip-lobster.exe`。

### D. Agent 引擎 + Redis（原有栈）

```powershell
docker compose up -d
# 或见根 README.md 与 src/agent/README.md
```
