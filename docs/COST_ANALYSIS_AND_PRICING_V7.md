## 六、为什么底线设在 ¥1,980？

```
理由1：成本覆盖安全边际
  单席AI成本 ¥784 + 缓冲 ¥100 = ¥884 可变成本
  ¥1,980 ÷ ¥884 = 2.24倍，即使规模未达盈亏平衡，每席仍有正边际
  固定成本摊薄后：1000席时固定摊薄¥281，单席总成本¥1,165，¥1,980毛利率41%

理由2：保护渠道定价秩序
  底线价¥1,980，代理向下游至少按¥2,980卖（62折档）
  代理每席至少赚¥1,000毛利，有动力推广
  不会出现代理恶性竞争（互相低价抢客）

理由3：品牌定位
  ¥1,980/账号/月 = 约 ¥66/天
  传统代运营单账号 ¥3,000-¥8,000/月
  ¥1,980 在大批量时仍显"超值"，不显廉价

理由4：V7之前 V6 的¥1,480 太低
  ¥1,480/席，AI成本¥884，边际贡献仅¥596
  需要472席才能覆盖固定成本，风险太高
  ¥1,980/席，边际贡献¥1,096，只需257席覆盖，更安全
```
