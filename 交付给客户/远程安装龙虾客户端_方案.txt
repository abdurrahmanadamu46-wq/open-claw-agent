【远程把龙虾装在客户电脑上】可行方案

一、前提
  - 客户电脑能联网（装好后连总控用内网 192.168.1.x 或您指定的总控地址）。
  - 您能远程操作或指导客户：远程桌面 / 向日葵 / ToDesk / 微信视频 等。

二、方案 A：远程桌面直接装（推荐，可控度高）
  1. 用向日葵、ToDesk、Windows 远程桌面等连上客户电脑。
  2. 把「客户包_免exe」打成 zip（或已有压缩包），通过 U 盘/网盘/微信/QQ 传到客户机，或本机共享文件夹让客户机访问。
  3. 在客户机上解压到固定目录，例如 D:\LobsterClient。
  4. 用记事本打开解压目录里的 .env.vip，修改：
       C_AND_C_SERVER_URL=http://总控IP:3000/agent-cc
       CLIENT_DEVICE_TOKEN=客户侧提供的 token（若有）
       MACHINE_CODE=VIP-CLIENT-001（或按客户编号）
  5. 双击「启动龙虾客户端.bat」，确认窗口出现 🟢 连接成功。
  6. 可选：把「启动龙虾客户端.bat」拖到桌面/发送到桌面快捷方式，方便客户日后使用。
  优点：配置准确、可当场验证；缺点：需要客户配合一次远程。

三、方案 B：发安装包 + 远程指导（客户自己点）
  1. 您把「客户包_免exe」zip + 一份《客户安装说明》（见下方模板）发给客户。
  2. 约时间远程（语音/视频），指导客户：解压 → 编辑 .env.vip 填总控 IP → 双击启动龙虾客户端.bat。
  3. 客户把连接结果（🟢 或报错截图）发您确认。
  优点：不一定要远程控制权限；缺点：依赖客户操作，容易填错。

四、方案 C：半自动安装脚本（适合多台客户机）
  1. 客户包内已提供「安装_可选总控IP.bat」：解压后运行即可写入总控地址并创建桌面快捷方式。
  2. 用法：把「客户包_免exe」发给客户，让客户解压到任意目录（如 D:\LobsterClient），然后：
       - 双击「安装_可选总控IP.bat」：按默认总控 IP 192.168.1.7 写入并创建快捷方式；
       - 或 在命令行运行：安装_可选总控IP.bat 192.168.2.100（把 192.168.2.100 换成实际总控 IP）。
  3. 脚本会只修改 .env.vip 里的 C_AND_C_SERVER_URL，保留 CLIENT_DEVICE_TOKEN、MACHINE_CODE；并在桌面创建「启动龙虾客户端」快捷方式。
  4. 客户日后双击桌面快捷方式或「启动龙虾客户端.bat」即可。
  优点：多台机器可复用，总控 IP 由您远程报给客户或命令行传入；缺点：客户需先解压再运行该 bat。

五、方案 D：一键安装器 exe（进阶）
  1. 用 Inno Setup / NSIS 等把「客户包_免exe」+ Node 运行时（或绿色 node）打成安装程序。
  2. 安装向导中让客户选择或输入「总控地址」，安装时写入 .env.vip。
  3. 安装完成后桌面生成「启动龙虾」快捷方式。
  优点：客户体验像正规软件；缺点：打包与维护成本较高。

六、建议
  - 客户数量少、首次交付：用方案 A（远程直接装），装完当场验 🟢。
  - 客户能自己解压改配置：用方案 B + 一份简短《客户安装说明》。
  - 客户数量多、总控 IP 固定：在方案 A 基础上加方案 C，做一个 install.bat 统一部署。

七、《客户安装说明》简短模板（供复制给客户）
  —————————————————————
  1. 解压「龙虾客户端」压缩包到任意文件夹（例如 D:\LobsterClient）。
