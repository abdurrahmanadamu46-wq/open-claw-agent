========================================
  VIP 一号客户交付包 — 请先看这里
========================================

【网址】
打开「公网总控网址.txt」，里面的地址即当前总控公网网址：
  https://loud-feet-rescue.loca.lt

【.exe】
1. 双击运行「生成exe.bat」，等待打包完成（首次会从 GitHub 下载约 50MB，国内可能很慢或卡住）。
2. 若卡在 0%% 或闪退：请打开「打包exe_卡住或闪退时看这里.txt」，可改用「客户包_免exe」立刻交付（客户需装 Node.js，双击 启动龙虾客户端.bat 即可）。
3. 若打包成功：本文件夹会出现 vip-lobster.exe，把「vip-lobster.exe」和「.env.vip」一起发给客户即可。

【使用说明】
详见「使用说明.txt」— 您本机需保持总控后端 + localtunnel 运行，客户才能连上。
